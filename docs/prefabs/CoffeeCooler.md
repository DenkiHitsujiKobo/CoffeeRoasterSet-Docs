# CoffeeCooler

焙煎したコーヒー豆を冷やすための冷却器です。

WIP: ここに画像を載せる


## 使いかた

- ワールドに固定で設置します。
- [Colander]を近くで手放すと、真上に乗るよう位置を補正します。

{: .highlight }
現在のところ、実際に豆の温度などを管理する機能は本アセットに実装されていません。  
コーヒー焙煎の過程を楽しむ演出としてご利用ください。


## 設定項目

以下のコンポーネントが設定できます。  
詳細は各コンポーネントの解説ページをご覧ください。

- RoasterBase/SnapPoint_colander: [PickupSnapPoint]
