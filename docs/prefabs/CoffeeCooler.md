# CoffeeCooler

焙煎したコーヒー豆を冷やすための冷却器です。


## 使いかた

- ワールドに固定で設置します。
- [Colander]を近くで手放すと、真上に乗るよう位置を補正します。


## 設定項目

以下のコンポーネントが設定できます。  
詳細は各コンポーネントの解説ページをご覧ください。

- RoasterBase/SnapPoint_colander: [PickupSnapPoint]
