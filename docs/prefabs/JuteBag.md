# JuteBag

生のコーヒー豆がつまった麻袋です。


## 使いかた

- Pickupオブジェクトです。UseすることでパーティクルのON/OFFを切り替えます。
- ほかの容器オブジェクトにパーティクルを当てることで、生のコーヒー豆を渡すことができます。
  - 袋の中身がなくなることはありません。
